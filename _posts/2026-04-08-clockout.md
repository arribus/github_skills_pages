In this organization, we work a maximum of **7 hours net** per day. However, since **1h 15m** is automatically deducted for breaks, I need to stay for a total of **8h 15m**.

### Calculate Your Departure
Enter your arrival time below to see exactly when you should leave.

<div style="background: #f9f9f9; padding: 20px; border-radius: 8px; border: 1px solid #ddd;">
    <label>Arrival Time:</label>
    <input type="time" id="arrival" style="padding: 8px; border-radius: 4px; border: 1px solid #ccc;">
    <p style="margin-top: 15px;">You should leave at: <strong id="departure">--:--</strong></p>
</div>

<script>
    const input = document.getElementById('arrival');
    const output = document.getElementById('departure');

    input.addEventListener('input', () => {
        const [hrs, mins] = input.value.split(':').map(Number);
        
        // Add 8 hours and 15 minutes
        let depHrs = hrs + 8;
        let depMins = mins + 15;

        if (depMins >= 60) {
            depMins -= 60;
            depHrs += 1;
        }
        
        const displayHrs = String(depHrs % 24).padStart(2, '0');
        const displayMins = String(depMins).padStart(2, '0');

        output.innerText = `${displayHrs}:${displayMins}`;
    });
</script>

---
*Note: This calculation includes the mandatory 1h15m deduction.*
