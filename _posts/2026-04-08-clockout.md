To comply with the organization's rule of **7 hours maximum** (with a mandatory **1h 15m deduction**), I have built this quick calculator.

### Calculate Departure
Enter the time you arrived at the organization:

<div style="background-color: #f6f8fa; padding: 20px; border-radius: 6px; border: 1px solid #d0d7de; max-width: 300px;">
  <label for="arrival" style="display: block; margin-bottom: 10px; font-weight: bold;">Arrival Time (24h):</label>
  <input type="time" id="arrival" style="width: 100%; padding: 8px; border-radius: 4px; border: 1px solid #8c959f;">
  
  <p style="margin-top: 20px; font-size: 1.1em;">
    Leave at: <span id="departure" style="font-weight: bold; color: #0969da;">--:--</span>
  </p>
</div>

<script>
  const arrivalInput = document.getElementById('arrival');
  const departureOutput = document.getElementById('departure');

  arrivalInput.addEventListener('input', function() {
    if (!this.value) return;

    const [hours, minutes] = this.value.split(':').map(Number);

    // LOGIC: 
    // Target Work: 7 hours
    // Deduction: 1 hour 15 minutes
    // Total Presence Required: 8 hours 15 minutes
    
    let totalMinutes = minutes + 15;
    let totalHours = hours + 8;

    // Handle minute overflow
    if (totalMinutes >= 60) {
      totalMinutes -= 60;
      totalHours += 1;
    }

    // Handle 24-hour format rollover
    if (totalHours >= 24) {
      totalHours -= 24;
    }

    const finalH = String(totalHours).padStart(2, '0');
    const finalM = String(totalMinutes).padStart(2, '0');

    departureOutput.innerText = `${finalH}:${finalM}`;
  });
</script>

---
**How it works:**
The script takes your arrival time and adds **8 hours and 15 minutes**. This ensures that after the organization's automated system deducts the 1h15m, you have exactly 7 hours of "work time" registered.
